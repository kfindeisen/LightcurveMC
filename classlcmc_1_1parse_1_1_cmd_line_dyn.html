Detailed Description

Command-line parsing class that uses dynamically allocated arguments rather than statically allocated ones.

Otherwise, its behavior is identical to TCLAP::CmdLine.

This class is final.

Invariant
The memory for all arguments in CmdLineDyn is managed by the CmdLineDyn object.

Adds multiple arguments with a xor constraint.

Parameters
[in]xorsA list of TCLAP::Arg to add.
Precondition
Each element of xors was allocated using the new operator
No element of xors is being managed by a smart pointer or other object
Postcondition
Each element of xors is now an argument on the command line
Each element of xors will be deallocated when this is destroyed
Unless exactly one of the arguments in xors is given on the command line, parse() will throw an exception.
Exceptions
