lcmc::test::StatData::StatData ( )

Defines the data for each test case.

Precondition
A text file called ptfjds.txt exists in the working directory and contains a list of Julian dates.
Exceptions
kpfutils::except::FileIoThrown if ptfjds.txt could not be opened or has the wrong format.
std::bad_allocThrown if there is not enough memory to store the times.

Object construction is atomic.

Member Data Documentation

vector<double> lcmc::test::StatData::ptfTimes

Represents sampling at the irregular PTF cadence.

This cadence has several features, particularly a long gap between the 2009 and 2010 seasons, that tends to introduce artifacts in statistical analyses.

vector<double> lcmc::test::StatData::unifTimes

Represents sampling at a uniform daily cadence for 100 days.

Statistical tests using this cadence should be well-behaved.
