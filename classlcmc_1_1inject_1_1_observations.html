Detailed Description

Class for generating data samples for signal injection testing.

Objects of class Observations represent individual sources drawn from a selected sample. Observations objects are non-deterministic in the sense that the user only specifies which sample the light curve comes from, but not which light curve from within the sample gets used. However, any individual object is immutable, and can be can be used without worrying about determinism.

Invariant
Observations is immutable.
The light curve has a median flux of 1, so that synthetic light curve amplitudes can be expressed in units of the source flux.
