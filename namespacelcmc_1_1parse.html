Adds a parameter to a light curve specification if the appropriate command line argument was used.

Parameters
[in]rangethe RangeList to update
[in]paramNamethe name of the parameter
[in]paramValuethe command-line argument to translate into a range
[in]distribthe distribution values follow within the range
Postcondition
If paramValue was specified on the command line, the corresponding parameter range is added to range. Otherwise, range is unchanged.
Exceptions
lcmc::utils::except::UnexpectedNanThrown if either element of paramValue is NaN
lcmc::stats::except::ExtraParamThrown if a value for the parameter is already in the list
lcmc::stats::except::NegativeRangeThrown if paramValue.second > paramValue.first
std::invalid_argumentThrown if distrib is not a valid value.
std::bad_allocThrown if there is not enough memory to add an element to the list.
Exception Guarantee
The arguments are unchanged in the event of an exception.

Converts the input arguments to a set of variables.

Parameters
[in]argc,argvan array of C-strings denoting the arguments, using the C convention
[out]sigmathe root-mean-square amount of noise to add to all flux measurements
[out]nTrialsthe number of simulations to execute per bin
[out]toPrintif > 0, the program will print the first few light curves generated. This setting is intended primarily for debugging.
[out]paramRangesthe minimum and maximum values of light curve parameters allowed by the user
[out]jdListthe name of a file containing the Julian dates of the simulated observations
[out]lcNameListthe list of light curve types to simulate, one after the other
[out]lcListthe list of light curve types to simulate
[out]statListthe list of statistics to calculate for each simulated light curve
[out]dataSetthe name of a file containing observed light curves into which to inject simulated light curves
[out]injectModeif true, the program will carry out an injection analysis rather than merely generating theoretical light curves.
Exceptions
std::bad_allocThrown if there is not enough memory to represent the command line arguments.
std::logic_errorThrown if a bug is detected in the code.
lcmc::parse::except::ParseErrorThrown if the output cannot be parsed.
Exception Guarantee
All arguments are left in valid states in the event of an exception.
