Calculates the autocorrelation function for a time series.

Parameters
[in]timesTimes at which data were taken
[in]dataThe data (typically fluxes or magnitudes) measured at each time
[in]deltaT,nAcfThe spacing and number of grid points over which the ACF should be calculated. The grid will run from lags of 0 to (nAcf-1)*deltaT.
[out]acfThe value of the autocorrelation function at each offset.
Precondition
times contains at least two unique values
times is sorted in ascending order
data.size() = times.size()
data[i] is the measurement taken at times[i], for all i
neither times nor data contains NaNs
deltaT > 0.0
nAcf > 0
Postcondition
the data previously in acf are erased
acf.size() = nAcf
acf[i] is an estimator of the autocorrelation function evaluated at Δt = i*deltaT, for all i
acf[i] = 0 if i*deltaT > max(times) - min(times)
Performance
O(F2) worst-case time, where F = ceil((max(times)-min(times))/deltaT)
O(F log F) best-case time
Exceptions
lcmc::utils::except::UnexpectedNanThrown if there are any NaN values present in times or data.
lcmc::stats::except::NotEnoughDataThrown if times and data do not have at least two values.
std::invalid_argumentThrown if times and data do not have the same length or if deltaT or nAcf are not positive.
std::runtime_errorThrown if the internal calculations produce an error.
std::bad_allocThrown if there is not enough memory to compute the autocorrelation function
Exception Guarantee
The function parameters are unchanged in the event of an exception.

Generates a C array of evenly spaced values.

Parameters
[in]minThe minimum value in the grid
[in]maxAn upper limit on the grid
[in]stepThe grid spacing
[out]grid,nA newly allocated array containing the grid.
Precondition
min < max
step > 0
Postcondition
min = min(grid)
max - step ≤ max(grid) < max
for all i ∈ [0, n-2], grid[i+1] - grid[i] = step
Exceptions
std::invalid_argumentThrown if max ≤ min or step ≤ 0
std::bad_allocThrown if there is not enough memory to construct grid.
