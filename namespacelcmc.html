models::ParamList lcmc::drawParams ( const models::RangeList &  limits)

Randomly generates parameter values within the specified limits.

Parameters
[in]limitsA RangeList giving the parameters to generate, the minimum and maximum values for each, and the distribution from which to sample
Returns
A ParamList giving a value for each parameter
Postcondition
X is a parameter in the return value if and only if X is a parameter in limits
for any X in the return value, limits.getMin(X) ≤ return.get(X) ≤ limits.getMax(X)
No element of the return value is NaN.
Exceptions
std::bad_allocThrown if not enough memory to generate random values.
std::logic_errorThrown if drawParams() does not support all distributions in limits
Exception Guarantee
Function parameters are unchanged in the event of an exception.
shared_ptr< RInside > lcmc::getRInstance ( )

Returns a well-defined reference to an R instance.

Returns
A pointer to a running R session.
Invariant
All pointers returned by getRInstance() refer to the same R instance.
Note
This behavior is forced by the R API: it is impossible for a single process to open two consecutive R sessions.
Exceptions
std::bad_allocThrown if there is not enough memory to start the process.
lcmc::except::BadRSessionThrown if R could not be started due to an internal error in R.
Exception Guarantee
The program state is unchanged in the event of an exception.

vector< double > &  times,
vector< double > &  baseFlux 
)

Generates an observed light curve for an injection analysis.

Parameters
[in]catalogThe name of a file containing a list of light curves to sample from.
[out]times,baseFluxStore a light curve randomly selected from catalog.
Postcondition
The data previously in times and baseFlux are erased
times.size() = baseFlux.size()
The fluxes in the original light curve are scaled to a median flux of one, then offset to a median flux of zero.
Exceptions
lcmc::inject::except::NoCatalogThrown if catalog does not exist.
kpfutils::except::FileIoThrown if the catalog or the light curve could not be read.
std::bad_allocThrown if there is not enough memory to store the output.
Exception Guarantee
The program is in a valid state in the event of an exception.

void lcmc::simLightCurve ( const models::LightCurveType &  curve,
const models::ParamList &  params,
const vector< double > &  times,
const vector< double > &  noise,
vector< double > &  lcFluxes 
)

Generates a random light curve, incorporating all the simulation settings.

Parameters
[in]curveThe type of light curve to generate
[in]paramsThe parameters for the light curve.
[in]timesThe times at which the light curve will be observed.
[in]noiseThe contaminating signal, in units of the median source flux, to add to the generated light curve.
[out]lcFluxesThe generated fluxes, including noise.
Precondition
params contains a valid entry for each parameter required by curve
times.size() > 0
times.size() = noise.size()
Postcondition
Any data previously in lcFluxes is erased
lcFluxes.size() = times.size()
lcFluxes[i] = signal(times[i]) + noise[i]
Exceptions
std::bad_allocThrown if there is not enough memory to calculate the light curve.
std::logic_errorThrown if a bug is found in the flux calculations.
std::invalid_argumentThrown if curve is invalid.
lcmc::models::except::MissingParamThrown if a required parameter is missing from params.
lcmc::models::except::BadParamThrown if params contains an illegal parameter value.
Exception Guarantee
The program is in a consistent state in the event of an exception.
