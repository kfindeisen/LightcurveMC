Detailed Description

GaussianProcess represents variables that vary as any kind of Gaussian process in magnitude space.

GaussianProcess is an abstract base class. Subclasses of GaussianProcess represent Gaussian processes with specific kernels by implementing the private function getCovar() and optionally overriding the private function solveFluxes().
