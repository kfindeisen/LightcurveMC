std::istream& operator>> ( std::istream &  str,
std::pair< double, double > &  val 
)

Extracts formatted input into a pair of double precision floating point values.

The string is assumed to be formatted as two real literals, separated by whitespace.

Parameters
[in]strThe stream to read from.
[out]valThe pair in which to store the values.
Returns
A reference to str.
Exceptions
std::ios_base::failureThrown if the input cannot be read and str.exceptions() is set to throw exceptions. If thrown, str remains in a valid state.
