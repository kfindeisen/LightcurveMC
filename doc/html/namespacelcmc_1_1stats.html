Calculates the light curve amplitude.

The amplitude is defined as the range between the 5th and 95th percentiles. This definition is relatively insensitive to outliers.

double lcmc::stats::getC1 ( const DoubleVec &  mags)

Calculates the modified C1 statistic.

The statistic is identical to that presented in Grankin (2007), except that the minimum and maximum magnitude have been replaced with the 5th and 95th percentile, respectively. This change makes the C1 statistic much less sensitive to outliers.

Parameters
[in]magsA vector of magnitudes from which to calculate C1.
Returns
The C1 statistic.
Precondition
mags contains at least three finite values
mags contains at least two distinct finite values
mags may contain NaNs

Computes the fraction of pairs of magnitudes above some threshold found in each Δt bin of a ΔmΔt plot.

Parameters
[in]deltaTThe time differences of the ΔmΔt plot.
[in]deltaMThe magnitude differences of the ΔmΔt plot.
[in]binEdgesA vector containing the (N+1) boundaries of the N bins in which to count high-Δm pairs.
[out]fracsA vector containing the fraction of Δm values in each bin that exceed threshold.
[in]thresholdThe characteristic magnitude difference above which Δm values are to be counted.
Precondition
deltaT is sorted in ascending order

deltaT does not contain any NaNs
deltaM does not contain any NaNs
binEdges does not contain any NaNs
Postcondition
fracs.size() = binEdges.size() - 1
For all i ∈ [0, binEdges.size()-1], fracs[i] contains the fraction of deltaM > threshold where deltaT ∈ [binEdges[i], binEdges[i+1]).

Calculates the RMS binned over ever-larger subintervals of the data.

rmsVsTAllPairs() considers all subintervals of the data set

Parameters
[in]timesTimes at which data were taken
[in]fluxesFlux measurements of a source
[out]timeStepsA list of the lengths of all subintervals
[out]rmsValuesA list of the RMS over each subinterval
Precondition
times contains at least two unique values
times is sorted in ascending order
fluxes is of the same length as times
fluxes[i] is the flux of the source at times[i], for all i
times does not contain NaNs
fluxes does not contain NaNs
intervals does not contain NaNs
Postcondition
intervals is sorted in ascending order
if N == length(times), length(intervals) == length(rmsValues) == (N C 2)
Exceptions
invalid_argumentThrown if the preconditions on times or length(fluxes) are violated.
Performance:
O(times.size()^3) time

Reduce to O(times.size()^2) time through dynamic programming
