Detailed Description

A RangeList contains the minimum and maximum arguments to consider for the light curve.

The minimum and maximum argument values are used to generate a suitable ParamList for use in simulation.

Member Enumeration Documentation

enum lcmcmodels::RangeList::RangeType

Represents the distribution from which a parameter is to be drawn.

Enumerator
UNIFORM 

Indicates a parameter is drawn uniformly from the interval [minimum, maximum).

LOGUNIFORM 

Indicates a parameter is drawn log-uniformly from the interval [minimum, maximum).
