Detailed Description

ILightCurve is the interface class for all light curve models.

Each class implementing ILightCurve represents a particular kind of light curve, and each ILightCurve object represents a single realization of that type of light curve. No application using the LightCurves should need more than the interface provided by ILightCurve.

A LightCurve object represents a simulated set of observations, with fluxes determined by the model parameters and times determined by the desired cadence of the mock data set.

Invariant
ILightCurve is immutable.
A flux of zero represents a baseline flux. The exact significance may depend on the type of light curve, but in general should be near either the mean or the median of the light curve.
For no parameters and at no value of time does flux() return a number less than -1.
