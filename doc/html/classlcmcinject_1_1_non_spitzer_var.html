Detailed Description

Data generator for sources that are not IR excess sources in Spitzer and variables in PTF.

Constructor & Destructor Documentation

lcmcinject::NonSpitzerVar::NonSpitzerVar ( )

Initializes the object to represent a randomly selected light curve meeting the sample criteria.

Precondition
There exists a text file named "nonspitzervar.cat" in the working directory. Each line in the file is the path to a text file containing the light curve of a source in thesample.
Exceptions
FileIoThrown if either the catalog file or the selected light curve file do not exist.
