Detailed Description

LightCurveType specifies which LightCurve should be created for a particular simulation.

Each allowed value corresponds to exactly one light curve model.

Note
Since forward declaring enums requires a compiler that supports C++11, and I don't want to assume that yet, define LightCurveType as a class supporting no operation other than equality. Actual LightCurveType constants are only visible to those functions that need them.
